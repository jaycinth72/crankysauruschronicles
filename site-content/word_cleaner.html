Word HTML Cleaner

Clean up Microsoft Word HTML exports for your blog posts

How to Use:

  1. Create your blog post in Microsoft Word
  2. Save as "Web Page, Filtered (*.html)"
  3. Open the HTML file in a text editor and copy all the content
  4. Paste it in the "Raw HTML from Word" box below
  5. Click "Clean HTML" to get blog-ready content
  6. Copy the cleaned HTML and save it as your post's content file
